# Crude-Volatility_Smile-Visualisation
Short Visualisation tool working to show the smile of a volatility surface of a certain underlying (Brent and WTI here)
Takes bloomberg data as CSV and create different DF and Plots to look at the smile of different contracts/underlyings. Said CSV is included in the repository for example
